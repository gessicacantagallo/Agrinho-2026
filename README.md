# 🌾 Agrinho 2026 - Agricultura Sustentável

Site moderno e responsivo sobre práticas de agricultura sustentável, desenvolvido como projeto educacional para promover a sustentabilidade agrícola.

## 📋 Características

✅ **Design Responsivo** - Funciona perfeitamente em desktop, tablet e mobile  
✅ **Interface Intuitiva** - Navegação clara e fácil de usar  
✅ **Animações Suaves** - Efeitos visuais modernos e fluidos  
✅ **Menu Mobile** - Hamburger menu adaptativo  
✅ **Formulário de Contato** - Validação de dados e feedback visual  
✅ **SEO Otimizado** - Meta tags e estrutura semântica  
✅ **Performance** - Carregamento rápido e otimizado  

## 🎯 Seções do Site

### 1. **Navegação (Navbar)**
- Logo com ícone
- Links para todas as seções
- Menu responsivo para mobile

### 2. **Hero Section**
- Imagem inspiradora
- Texto de boas-vindas
- Chamada para ação (CTA)

### 3. **Sobre Agricultura Sustentável**
4 cards explicativos com:
- Sustentabilidade Ambiental
- Impacto Social

- Preservação Ecológica

### 4. **Práticas Sustentáveis**
6 cards detalhados:
- 💧 Irrigação Eficiente
- 🌱 Rotação de Culturas
- ♻️ Compostagem
- 🌳 Agroflorestas
- 🐛 Controle Biológico
- 🔄 Economia Circular

### 5. **Benefícios**
- Estatísticas visuais
- Lista de vantagens
- Impacto positivo

### 6. **Contato**
- Formulário de contato funcional
- Informações de contato
- Links de redes sociais

### 7. **Footer**
- Copyright
- Créditos

## 🚀 Como Usar

1. **Clone o repositório**
   ```bash
   git clone https://github.com/gessicacantagallo/Agrinho-2026.git
   cd Agrinho-2026
   ```

2. **Abra o arquivo index.html**
   - Duplo clique em `index.html` para abrir no navegador
   - Ou use um servidor local (VS Code Live Server, Python, etc.)

3. **Explore o site**
   - Navegue pelas diferentes seções
   - Teste o menu mobile
   - Interaja com o formulário de contato
